# AIFFEL_quest_rs

아이펠 리서처 과정 19기 퀘스트 제출 및 학습 기록을 관리하기 위한 저장소입니다.

이 저장소는 과정 중 진행하는 Main Quest, Exploration, GoingDeeper 과제를 정리하기 위해 사용합니다.

## Folder Description

- `MainQuest`: 메인 퀘스트 제출 폴더
- `Exploration`: Exploration 과제 제출 폴더
- `GoingDeeper`: GoingDeeper 과제 제출 폴더

## Notes

각 퀘스트 폴더에는 과제 노트북 파일과 Peer Review Template 기반의 `README.md` 파일을 포함합니다.
